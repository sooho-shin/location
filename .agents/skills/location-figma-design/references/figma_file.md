## Usage

- LOCATION 앱 Figma 디자인 작업은 기본적으로 이 Figma 프로젝트를 작업 위치로 간주한다.
- 특별한 요청이 없으면 `Primary Design File`에 기록된 design file에 작업한다.
- 새 Figma design file이 필요하면 이 프로젝트 안에 만든다.
- 사용자가 별도 design file URL을 제공하면 그 파일을 우선한다.
- OAuth 토큰, API 키, 인증 정보는 저장하지 않는다.
